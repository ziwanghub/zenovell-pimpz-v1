# PHASE 6D — SECTION 4 MOBILE TYPOGRAPHY MICRO PASS CHANGELOG

## Scope

This changelog records typography-only refinements applied during the Section 4 Mobile Typography Micro Pass.

No layout, hierarchy, CTA, image, or business-logic changes were made.

---

## Delta

### 1. Thai Font Rendering Quality Improved

- Thai text rendering was moved to a more stable font path for improved mark placement and cleaner glyph presentation
- The change was applied to intro and in-card Thai text only

### 2. Intro Heading Typography Stabilized

- Intro heading typography was refined to reduce visual noise
- Premium calm and first-scan stability were prioritized over visual size

### 3. Intro Subtitle Comfort Improved

- Subtitle typography was adjusted for easier first-scan reading
- The hierarchy remains clearly subordinate to the heading

### 4. Card Fit Line Readability Improved

- Fit-line text at small widths received a readability-focused micro pass
- Line-height behavior was adjusted to reduce compression at `360` and `375`

### 5. Difference Line Scan Speed Improved

- Difference-line typography was clarified while preserving subordinate position under Fit
- The goal was faster scan, not stronger dominance

### 6. Support Text Comfort Improved

- Small Thai support text was refined for better comfort at small widths
- Support text remains intentionally lower priority than decision layers

### 7. Trust Text Readability Improved

- Trust text was lightly refined for legibility only
- No increase in trust-layer dominance was introduced
